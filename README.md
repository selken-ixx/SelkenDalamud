# Selken Dalamud

Selken Dalamud 是 Final Fantasy XIV / Dalamud 自定义插件发布仓库。

本仓库用于提供自定义插件源、发布信息和插件下载入口。插件以 GitHub Release 形式发布，不提交到 Dalamud 官方插件库。

## 插件源

```text
https://raw.githubusercontent.com/selken-ixx/SelkenDalamud/main/pluginmaster.json
```

## 更新提示

插件更新通过上方自定义插件源和 GitHub Release 分发。若插件安装器刷新后仍显示旧版本，通常是本地插件列表缓存尚未重载；可以尝试关闭并重新打开插件安装器、重新刷新插件源，或重载插件后再查看版本。

## 数据说明

插件数据默认保存在用户本机。除非用户主动导出、复制或上传，否则插件不会上传角色、库存或雇员数据。
